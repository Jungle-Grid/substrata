# Roadmap

## Current MVP

Source package upload, facts, candidate review paths, company history comparison, memo drafts, human-review workflow, and audit trail.

## Near-term improvements

- Better PDF and structured-document parsing
- Clearer provider health and execution diagnostics
- More review-path-specific evidence presentation

## Company history import

Expand import formats and normalization while retaining record-scoped provenance, organization boundaries, and reviewer control.

## Reviewer collaboration

Add assignment, comments, version comparison, approval scopes, and clearer queue operations.

## Export and download workflows

Improve memo export, evidence package download, and internal review handoff formats.

## Enterprise controls

Priorities include SSO, retention/deletion controls, managed storage, stronger audit immutability, and configurable workspace governance.

## Integrations

Potential integrations include document repositories, product-information systems, ticketing, and controlled remote execution providers.

## Long-term vision

Make evidence-backed, human-approved classification workups reusable across product families and recurring technical reviews.
